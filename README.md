Лабораторна робота №4: Реалізація породжуючих патернів проектування на JavaScript

## Реалізовані патерни

1. **Singleton (Одинак)** - забезпечує існування лише одного екземпляра класу
2. **Factory Method (Фабричний метод)** - створює об'єкти без вказівки точного класу
3. **Abstract Factory (Абстрактна фабрика)** - створює сімейства пов'язаних об'єктів
4. **Builder (Будівельник)** - покрокове конструювання складних об'єктів
5. **Prototype (Прототип)** - створення нових об'єктів шляхом клонування

## Структура проекту
├── singleton/ # Патерн Одинак
├── factory/ # Фабричний метод
├── abstractFactory/ # Абстрактна фабрика
├── builder/ # Будівельник
├── prototype/ # Прототип
└── index.html # Демонстрація всіх патернів


## Як запустити

1. Відкрийте файл `index.html` у браузері
2. Відкрийте консоль розробника (F12)
3. Переглянайте результати роботи кожного патерну

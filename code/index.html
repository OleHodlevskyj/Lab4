Породжуючі патерни проектування

Інструкція: Відкрийте консоль браузера (F12) для перегляду результатів роботи патернів.

1. Singleton (Одинак)

Гарантує, що клас має лише один екземпляр, та надає глобальну точку доступу до нього.

2. Factory Method (Фабричний метод)

Визначає інтерфейс для створення об'єкта, але залишає підкласам рішення про те, який клас інстанціювати.

3. Abstract Factory (Абстрактна фабрика)

Надає інтерфейс для створення сімейств пов'язаних або залежних об'єктів без вказівки їх конкретних класів.

4. Builder (Будівельник)

Відокремлює конструювання складного об'єкта від його представлення, дозволяючи одному процесу створювати різні представлення.

5. Prototype (Прототип)

Створює нові об'єкти шляхом клонування існуючого об'єкта-прототипу.
